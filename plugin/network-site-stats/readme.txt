Plugin giúp Super Admin theo dõi nhanh thống kê các site con trong mạng lưới WordPress Multisite.

== Description ==

Network Site Stats hiển thị bảng tổng quan các site trong mạng lưới bao gồm:
* ID site
* Tên site (Blog Name)
* Số bài viết, trang, bình luận
* Ngày bài viết mới nhất
* Trạng thái hoạt động

Plugin hoạt động ở chế độ **Network Activated** – chỉ Super Admin mới có quyền kích hoạt.

== Installation ==

1. Upload thư mục `network-site-stats` vào `/wp-content/plugins/`
2. Vào **Network Admin > Plugins**, tìm "Network Site Stats"
3. Nhấn **Network Activate**
4. Truy cập **Network Admin > Site Stats**
